个人收入支出管理系统
1.1 背景
个人日常财务管理需求普遍，需要一个简单易用的工具来记录收支并分析消费习惯。本系统旨在提供轻量级记账功能，支持多用户数据隔离，并通过可视化报表帮助用户了解财务状况。

1.2 目标
提供简洁的收支记录功能（金额、分类、备注）。

支持多用户注册与登录，用户数据完全隔离。

实现趋势分析、分类占比、月度对比三大报表。

系统管理员仅负责运维，不干预用户数据。

2. 用户角色
角色	职责描述
普通用户	记录个人收支、查看报表、管理分类。
系统管理员	负责运维（如用户禁用、查看系统日志），不接触用户数据。
3. 功能模块
3.1 前端模块
3.1.1 认证模块
登录：用户输入账号密码登录，获取JWT token。

注册：新用户注册，验证用户名唯一性。

3.1.2 收支管理模块
收支记录列表：展示所有记录，支持按日期筛选、分页。

新增/编辑收支：表单包含金额、类型（收入/支出）、分类、日期、备注。

分类管理：预设常用分类（餐饮、购物、工资等），用户可自定义。

3.1.3 报表分析模块
趋势分析：按日、周、月、年展示收支趋势（折线图）。

分类分析：展示收入/支出分类占比（饼图/环形图）。

月度对比：对比各月收支情况、结余变化（柱状图）。

3.1.4 个人设置
修改密码、个人信息。

3.2 后端模块
3.2.1 认证模块
提供登录、注册接口，生成JWT token，验证用户身份。

用户数据隔离：所有API通过用户ID过滤数据。

3.2.2 收支管理模块
收支记录CRUD接口，支持分页、日期筛选。

分类管理CRUD接口。

3.2.3 报表分析模块
趋势数据：根据日期范围返回每日/每周/每月收支汇总。

分类占比：返回指定时间段内各分类的收支总额。

月度对比：返回指定年份各月的收支及结余。

3.2.4 系统管理（运维）
提供管理员接口：用户列表查询、禁用用户、查看系统日志（仅管理员可访问）。

4. 技术栈
前端：Vue 3 + TypeScript + Vite + Element Plus + ECharts

后端：Node.js + Express + TypeScript + JWT + bcrypt

数据库：PostgreSQL 

部署：Docker + Nginx
